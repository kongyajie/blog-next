# 栈和队列概览

> 栈是一个后进先出的数据结构，JavaScript中没有栈，但可以用 Array 实现栈的所有功能。
> 栈的常见操作：`push` `pop` `stack[stack.length - 1]`

使用场景：
- 十进制转二进制（后出来的余数反而要排在前面）
- 判断字符串的括号是否有效
- 函数调用堆栈（最后调用的函数，最先执行完）
